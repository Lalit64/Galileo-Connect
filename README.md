This is a [Galileo Connect](https://nextjs.org/) project bootstrapped with [`Next.js`](https://nextjs.org).

## Getting Started

First install the dependencies: 

```bash
npm install
# or
yarn install
```

Then run the development server:

```bash
npm run dev
# or
yarn dev
```
Then... See the result!
